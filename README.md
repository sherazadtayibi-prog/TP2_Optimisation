# Projet d'optimisation de tournois de basketball

Ce projet compare trois approches d'optimisation:
- Programmation linéaire (PuLP)
- Algorithmes gloutons
- Programmation dynamique (récursivité + mémoisation)

# Objectif
L'objectif est de comparer leurs performances, leur complexité et la qualité des solutions obtenues.
